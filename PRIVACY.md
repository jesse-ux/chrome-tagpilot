# TagPilot 隐私政策

**最后更新时间**: 2025年1月14日  
**生效日期**: 2025年1月14日

---

## 概述

TagPilot ("我们")致力于保护用户隐私。本隐私政策说明了我们如何收集、使用和保护您的信息。

---

## 信息收集

### 1. 书签数据

- **收集内容**: 我们访问您浏览器的书签（标题、URL、文件夹结构）
- **使用目的**: 自动生成标签、搜索、整理书签
- **存储位置**: 所有数据仅存储在您的本地浏览器（Chrome Storage Local）
- **第三方共享**: 不会上传到我们的服务器

### 2. API 配置

- **收集内容**: 您提供的 LLM API Key 和 API Endpoint
- **使用目的**: 调用您配置的 AI 服务生成书签标签
- **存储位置**: 仅存储在您的本地浏览器
- **第三方共享**: 不会上传到我们的服务器

### 3. 网页内容

- **收集内容**: 当启用 AI 自动打标签时，会提取您收藏的网页标题和摘要
- **使用目的**: 发送到您配置的 LLM 服务生成标签
- **数据流向**: 直接发送到您配置的 AI 服务（OpenAI/Kimi/智谱/通义/火山等）
- **存储**: 我们不存储网页内容

---

## 数据使用

### TagPilot 不会：
- 将您的数据上传到我们的服务器
- 出售或出租您的任何信息
- 用于广告目的
- 未经同意与第三方共享您的数据

### TagPilot 会：
- 在本地浏览器存储书签标签和配置
- 调用您配置的 AI 服务生成标签（仅发送必要信息：URL、标题、页面摘要）
- 使用 Chrome Bookmarks API 读取和整理书签

---

## 权限说明

TagPilot 请求以下 Chrome 权限：

| 权限 | 用途 | 数据流向 |
|------|------|---------|
| `bookmarks` | 读取和管理书签 | 仅本地 |
| `storage` | 存储标签和配置 | 仅本地 |
| `tabs` | 获取当前页面信息 | 仅本地 |
| `activeTab` | 在当前页面显示标签浮层 | 仅本地 |
| `sidePanel` | 显示侧边栏界面 | 仅本地 |

### Host Permissions（主机权限）

- `https://api.openai.com/*` - 调用 OpenAI API
- `https://api.moonshot.cn/*` - 调用 Kimi API
- `https://open.bigmodel.cn/*` - 调用智谱 AI API
- `https://ark.cn-beijing.volces.com/*` - 调用火山引擎 API
- `https://dashscope.aliyuncs.com/*` - 调用通义千问 API
- `https://r.jina.ai/*` - 抓取网页摘要（CORS 代理）

**重要**: 我们仅作为客户端调用这些服务，不通过任何中间服务器。

---

## 数据安全

- **本地存储**: 所有配置和标签数据存储在您的浏览器中
- **加密传输**: 与 AI 服务的通信使用 HTTPS 加密
- **API Key 保护**: API Key 仅存储在本地，不会上传
- **开源透明**: 代码完全开源，可自行审查

---

## 第三方服务

当您启用 AI 自动打标签功能时，TagPilot 会将以下信息发送到您配置的 LLM 服务：

- 书签标题
- 书签 URL
- 网页摘要（通过 Jina.ai 提取）

**我们建议**:
- 使用信誉良好的 LLM 服务商
- 定期更换 API Key
- 阅读所选 LLM 服务的隐私政策

---

## 儿童隐私

TagPilot 不面向 13 岁以下儿童。如果我们发现意外收集了儿童信息，将立即删除。
